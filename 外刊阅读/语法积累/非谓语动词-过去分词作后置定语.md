# ## 定义
过去分词（done）放在名词后面，修饰该名词，表示被动或完成的含义。

## 核心用法
- 结构：名词 + done + by + 动作发出者
- 等价于：名词 + which/that + be + done

## ❌ 我的常见错误
- 把过去分词短语误判为句子的谓语动词
- 找不到真正的谓语，导致主干完全错误

## 📚 我的错题链接
- [[长难句积累-1#^2fca09]]：scenarios spun out by...，误把spun out当成谓语，真正谓语是get

## ✅ 解题技巧
看到done+by的结构，直接划掉，它一定是定语，不是谓语；然后继续往后找真正的谓语。
